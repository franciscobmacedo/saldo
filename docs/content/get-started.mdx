# Getting Started

Welcome to Saldo! This guide will help you quickly integrate Portuguese tax calculations into your application.

## Installation

```bash
npm install saldo
# or
pnpm add saldo
```

## Quick Start

<RunCode defaultCode={`
const result = simulateDependentWorker({
  income: 1500, // Monthly gross income in euros
  location: 'continent',
  married: false
});

console.log(\`Gross Income: €\${result.grossIncome.toFixed(2)}\`);
console.log(\`Taxable Income: €\${result.taxableIncome.toFixed(2)}\`);
console.log(\`Tax: €\${result.tax.toFixed(2)}\`);
console.log(\`Social Security: €\${result.socialSecurity.toFixed(2)}\`);
console.log(\`Net Salary: €\${result.netSalary.toFixed(2)}\`);
console.log(\`Yearly Net: €\${result.yearlyNetSalary.toFixed(2)}\`);`} />

## Understanding the Result

The `simulateDependentWorker` function returns a comprehensive breakdown of tax calculations:

```typescript
interface DependentWorkerResult {
  taxableIncome: number;        // Income subject to tax
  grossIncome: number;          // Total monthly gross income
  tax: number;                   // Monthly tax amount
  socialSecurity: number;       // Monthly social security contribution
  socialSecurityTax: number;   // Social security tax rate applied
  netSalary: number;            // Monthly net salary
  yearlyNetSalary: number;     // Annual net salary (14 months)
  yearlyGrossSalary: number;   // Annual gross salary (14 months)
  lunchAllowance: LunchAllowance; // Lunch allowance details
  bracket: {
    signal: "max" | "min";
    limit: number;
    max_marginal_rate: number;
    deduction: number;
    var1_deduction: number;
    var2_deduction: number;
    dependent_aditional_deduction: number;
    effective_mensal_rate: number;
  };
  taxRetentionTable: {
    situation: string;
    description: string;
    brackets: {
      signal: "max" | "min";
      limit: number;
      max_marginal_rate: number;
      deduction: number;
      var1_deduction: number;
      var2_deduction: number;
      dependent_aditional_deduction: number;
      effective_mensal_rate: number;
    }[];
    dependent_disabled_addition_deduction?: number;
  };
}
```

## Key Parameters

### Required Parameters

- **`income`**: Monthly gross income in euros

### Optional Parameters

- **`married`**: Whether the person is married (default: `false`)
- **`disabled`**: Whether the person has a disability (default: `false`)
- **`partnerDisabled`**: Whether the spouse has a disability (default: `false`)
- **`location`**: Tax region - `'continent'`, `'azores'`, or `'madeira'` (default: `'continent'`)
- **`numberOfHolders`**: Number of income holders in marriage - `1` or `2` (default: `null`)
- **`numberOfDependents`**: Number of dependents (default: `null`)
- **`numberOfDependentsDisabled`**: Number of dependents with disabilities (default: `null`)
- **`period`**: Tax calculation period in format "YYYY-MM-DD_YYYY-MM-DD" (default: `"2025-01-01_2025-07-31"`)
- **`socialSecurityTaxRate`**: Social security tax rate (default: `0.11`)
- **`twelfths`**: Holiday allowance distribution (default: `Twelfths.TWO_MONTHS`)
- **`lunchAllowanceDailyValue`**: Daily lunch allowance amount in euros (default: `10.2`)
- **`lunchAllowanceMode`**: Type of lunch allowance - `"cupon"` or `"salary"` (default: `"cupon"`)
- **`lunchAllowanceDaysCount`**: Working days per month for lunch allowance (default: `22`)

## Working with Types

The library exports all necessary types for TypeScript users:

```typescript
import { 
  simulateDependentWorker, 
  LocationT, 
  Twelfths, 
  DependentWorkerResult,
  SimulateDependentWorkerOptions,
  LunchAllowance 
} from 'saldo';

// Type-safe function
function calculateTax(options: SimulateDependentWorkerOptions): DependentWorkerResult {
  return simulateDependentWorker(options);
}

// Using location type
const location: LocationT = 'continent';

// Using the result type
const result: DependentWorkerResult = simulateDependentWorker({
  income: 1500,
  location
});
```

## Regional Differences

Each Portuguese region has different tax rates:

});

});

// Madeira
const madeira = simulateDependentWorker({
  income: 1500,
  location: 'madeira'
});

console.log('=== Regional Comparison (€1500 income) ===');
console.log(\`Continent - Net: €\${continent.netSalary.toFixed(2)}\`);
console.log(\`Azores - Net: €\${azores.netSalary.toFixed(2)}\`);
console.log(\`Madeira - Net: €\${madeira.netSalary.toFixed(2)}\`);`} />

## Error Handling

The library provides clear error messages for invalid configurations:

<RunCode defaultCode={`try {
  const result = simulateDependentWorker({
    income: 1500,
    married: true,
    // Missing numberOfHolders - required for married
  });
} catch (error) {
  console.error('Error:', error.message); // "'numberOfHolders' is required for married workers"
}`} />
