# Saldo

**Modern TypeScript library for Portuguese tax calculations**

Accurately calculate dependent worker payslips and independent worker (category B) income in Portugal with full TypeScript support.

## Quick Start

Choose the path you need:
- **Dependent workers** — Payroll with retention tables. Start with [dependent worker getting started](/docs/dependent/get-started).
- **Independent workers** — Simplified regime / RNH for self-employed. Start with [independent worker getting started](/docs/independent/get-started).

## Features

✅ **Dependent workers** - 2025 retention tables for continent, Azores, and Madeira  
✅ **Independent workers** - Simplified regime, expense caps, 248 business-day calendar  
✅ **Family Situations** - Single, married, dependents, disabilities  
✅ **Lunch Allowances** - Meal vouchers and cash allowances  
✅ **Holiday Bonuses** - 13th and 14th month distribution  
✅ **Detailed Tax Information** - Complete bracket and table data  
✅ **TypeScript Native** - Full type safety and IntelliSense  
✅ **Zero Dependencies** - Lightweight and reliable

## Core Concepts

### Tax Situations

The library automatically determines the appropriate tax table based on:
- **Marital Status**: Single or married
- **Number of Income Holders**: 1 or 2 (for married couples)
- **Dependents**: Number of dependent children/relatives
- **Disability Status**: Worker or partner disability benefits

### Supported Locations

- **`continent`**: Mainland Portugal
- **`azores`**: Azores region
- **`madeira`**: Madeira region

Each location has different tax rates and may have specific calculation rules.

### Time Periods

Tax calculations are based on specific date ranges as tax tables change over time. The library includes historical data for accurate calculations across different periods.
