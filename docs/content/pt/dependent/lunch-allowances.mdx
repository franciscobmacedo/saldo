# Subsídios de Refeição

Os subsídios de refeição são um benefício comum no emprego português. Compreender como são tributados é crucial para cálculos de salário precisos.

## Visão Geral

Os subsídios de refeição em Portugal têm tratamento fiscal específico:
- **Porção isenta**: Até certos limites diários
- **Porção tributável**: Valor que excede os limites
- **Segurança Social**: Pode estar isenta dependendo do tipo

> O tratamento de subsídio de refeição faz parte do simulador de **trabalhador por conta de outrem**. Os cálculos de trabalhador independente seguem regras de regime simplificado sem entradas de subsídio de refeição.

## Tipos de Subsídios de Refeição

### Cartões de Refeição (Cupões de Refeição)

Limite diário isento: **€10,20**

<RunCode defaultCode={`const result = simulateDependentWorker({
  income: 1500,
  lunchAllowanceDailyValue: 8.50,
  lunchAllowanceMode: "cupon",
  lunchAllowanceDaysCount: 22
});

console.log(\`Subsídio de refeição mensal: €\${result.lunchAllowance.gross.toFixed(2)}\`);
console.log(\`Porção isenta: €\${result.lunchAllowance.taxFree.toFixed(2)}\`);
console.log(\`Porção tributável: €\${result.lunchAllowance.taxable.toFixed(2)}\`);`} />

### Subsídio em Dinheiro (Em Dinheiro)

Limite diário isento: **€6,00**

<RunCode defaultCode={`const result = simulateDependentWorker({
  income: 1500,
  lunchAllowanceDailyValue: 7.50,
  lunchAllowanceMode: "salary",
  lunchAllowanceDaysCount: 21
});

console.log(\`Subsídio de refeição mensal: €\${result.lunchAllowance.gross.toFixed(2)}\`);
console.log(\`Porção isenta: €\${result.lunchAllowance.taxFree.toFixed(2)}\`);
console.log(\`Porção tributável: €\${result.lunchAllowance.taxable.toFixed(2)}\`);`} />

## Parâmetros

Ao usar subsídios de refeição, pode especificar estes parâmetros:

- **`lunchAllowanceDailyValue`**: Valor diário do subsídio em euros
- **`lunchAllowanceMode`**: Tipo de subsídio (`"cupon"` ou `"salary"`)
- **`lunchAllowanceDaysCount`**: Número de dias úteis por mês

## Cenários Comuns

### Cartões de Refeição Padrão

Cenário mais comum com cartões de refeição abaixo do limite isento:

<RunCode defaultCode={`const result = simulateDependentWorker({
  income: 1200,
  lunchAllowanceDailyValue: 7.50,
  lunchAllowanceMode: "cupon",
  lunchAllowanceDaysCount: 22
});

// Subsídio mensal: €165 (7.50 * 22)
// Isento: €165 (totalmente isento pois está abaixo de €10,20/dia)
// Tributável: €0
console.log('=== Cartões de Refeição Padrão ===');
console.log(\`Benefício líquido: €\${result.lunchAllowance.taxFree.toFixed(2)}\`);
console.log(\`Total do subsídio: €\${result.lunchAllowance.gross.toFixed(2)}\`);`} />

### Cartões de Refeição de Alto Valor

Cartões de refeição que excedem o limite isento:

<RunCode defaultCode={`const result = simulateDependentWorker({
  income: 1200,
  lunchAllowanceDailyValue: 12.00,
  lunchAllowanceMode: "cupon",
  lunchAllowanceDaysCount: 22
});

// Subsídio mensal: €264 (12.00 * 22)
// Isento: €224,40 (10.20 * 22)
// Tributável: €39,60 ((12.00 - 10.20) * 22)
console.log('=== Cartões de Refeição de Alto Valor ===');
console.log(\`Total do subsídio: €\${result.lunchAllowance.gross.toFixed(2)}\`);
console.log(\`Porção tributável: €\${result.lunchAllowance.taxable.toFixed(2)}\`);
console.log(\`Porção isenta: €\${result.lunchAllowance.taxFree.toFixed(2)}\`);`} />

### Subsídio de Refeição em Dinheiro

Subsídio em dinheiro com limite isento mais baixo:

<RunCode defaultCode={`const result = simulateDependentWorker({
  income: 1500,
  lunchAllowanceDailyValue: 8.00,
  lunchAllowanceMode: "salary",
  lunchAllowanceDaysCount: 21
});

// Subsídio mensal: €168 (8.00 * 21)
// Isento: €126 (6.00 * 21)
// Tributável: €42 ((8.00 - 6.00) * 21)
console.log('=== Subsídio de Refeição em Dinheiro ===');
console.log(\`Porção isenta: €\${result.lunchAllowance.taxFree.toFixed(2)}\`);
console.log(\`Porção tributável: €\${result.lunchAllowance.taxable.toFixed(2)}\`);`} />

### Sem Subsídio de Refeição

Usando a predefinição (sem subsídio):

<RunCode defaultCode={`const result = simulateDependentWorker({
  income: 1500
  // Nenhum parâmetro de subsídio de refeição especificado
});

## Implicações Fiscais

### Imposto sobre Rendimentos

- **Porção isenta**: Não incluída no rendimento tributável
- **Porção tributável**: Adicionada ao rendimento tributável mensal
- **Impacto**: Subsídios totais mais altos podem empurrá-lo para escalões fiscais mais altos

### Segurança Social

- **Cartões de refeição**: Geralmente isentos de contribuições de segurança social
- **Subsídios em dinheiro**: Podem estar sujeitos a segurança social (depende da implementação)

## Considerações Regionais

O tratamento fiscal do subsídio de refeição é geralmente consistente em:
- Portugal Continental (continente)
- Açores (azores)  
- Madeira (madeira)

No entanto, verifique sempre os regulamentos atuais, pois podem mudar.

## Boas Práticas

### Para Funcionários

1. **Compreenda o tipo de subsídio**: Cartões vs. dinheiro têm limites diferentes
2. **Acompanhe valores diários**: Garanta que está dentro dos limites isentos
3. **Considere a compensação total**: Equilibre salário base vs. subsídios

### Para Sistemas de Folha de Pagamento

1. **Valide limites diários**: Garanta que os cálculos usam valores isentos corretos
2. **Lide com meses parciais**: Ajuste a contagem de dias para feriados/ausências
3. **Mantenha-se atualizado**: Os limites isentos podem mudar anualmente
