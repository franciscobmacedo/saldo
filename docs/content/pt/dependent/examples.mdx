# Exemplos

Cenários de trabalhador por conta de outrem. Para casos de trabalhadores por conta própria, consulte os [exemplos de trabalhador independente](/independent/examples).

## Cálculos Básicos

### Trabalhador com Salário Mínimo

<RunCode defaultCode={`const minimumWageWorker = simulateDependentWorker({
  income: 760, // Salário mínimo português 2024
  location: 'continent'
});

console.log('=== Trabalhador com Salário Mínimo ===');
console.log(\`Rendimento Bruto: €\${minimumWageWorker.grossIncome.toFixed(2)}\`);
console.log(\`IRS: €\${minimumWageWorker.tax.toFixed(2)}\`);
console.log(\`Segurança Social: €\${minimumWageWorker.socialSecurity.toFixed(2)}\`);
console.log(\`Salário Líquido: €\${minimumWageWorker.net.salary.toFixed(2)}\`);
console.log(\`Líquido Anual: €\${minimumWageWorker.yearlyNetSalary.toFixed(2)}\`);`} />

## Situações Familiares

### Pai/Mãe Solteiro(a) com 2 Filhos

<RunCode defaultCode={`const singleParent = simulateDependentWorker({
  income: 1500,
  married: false,
  numberOfDependents: 2,
  location: 'continent'
});

console.log('=== Pai/Mãe Solteiro(a) (2 filhos) ===');
console.log(\`Líquido Mensal: €\${singleParent.net.salary.toFixed(2)}\`);

// Comparar com o mesmo rendimento mas sem dependentes
const noDependents = simulateDependentWorker({ 
  income: 1500, 
  location: 'continent' 
});

console.log(\`Poupança Fiscal com dependentes: €\${(noDependents.tax - singleParent.tax).toFixed(2)}\`);`} />

### Casado com Rendimentos Duplos (2 Titulares)

<RunCode defaultCode={`// Cônjuge 1
const spouse1 = simulateDependentWorker({
  income: 1400,
  married: true,
  numberOfHolders: 2,
  numberOfDependents: 1, // Um filho partilhado
  location: 'continent'
});

// Cônjuge 2
const spouse2 = simulateDependentWorker({
  income: 1100,
  married: true,
  numberOfHolders: 2,
  numberOfDependents: 0, // Filho já contado no cônjuge1
  location: 'continent'
});

const totalHouseholdNet = spouse1.net.salary + spouse2.net.salary;
console.log('=== Casado com Rendimentos Duplos ===');
console.log(\`Cônjuge 1 Líquido: €\${spouse1.net.salary.toFixed(2)}\`);
console.log(\`Cônjuge 2 Líquido: €\${spouse2.net.salary.toFixed(2)}\`);
console.log(\`Total do Agregado Líquido: €\${totalHouseholdNet.toFixed(2)}\`);`} />

## Exemplos Baseados em Períodos

### Transições de Períodos Fiscais

<RunCode defaultCode={`// Primeiro período de 2025
const firstPeriod = simulateDependentWorker({
  income: 1800,
  period: "2025-01-01_2025-07-31", // 1 de janeiro - 31 de julho
  location: 'continent'
});

// Segundo período de 2025
const secondPeriod = simulateDependentWorker({
  income: 1800,
  period: "2025-08-01_2025-09-30", // 1 de agosto - 30 de setembro
  location: 'continent'
});

// Terceiro período de 2025
const thirdPeriod = simulateDependentWorker({
  income: 1800,
  period: "2025-10-01_2025-12-31", // 1 de outubro - 31 de dezembro
  location: 'continent'
});

console.log('=== Comparação de Períodos Fiscais ===');
console.log(\`Primeiro Período Líquido: €\${firstPeriod.net.salary.toFixed(2)}\`);
console.log(\`Segundo Período Líquido: €\${secondPeriod.net.salary.toFixed(2)}\`);
console.log(\`Terceiro Período Líquido: €\${thirdPeriod.net.salary.toFixed(2)}\`);`} />

Estes exemplos demonstram a flexibilidade e natureza abrangente do Saldo para vários cenários fiscais portugueses.
