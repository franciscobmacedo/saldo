# Examples

Dependent worker scenarios. For self-employed cases, see the [independent worker examples](/independent/examples).

## Basic Calculations

### Minimum Wage Worker

<RunCode defaultCode={`const minimumWageWorker = simulateDependentWorker({
  income: 760, // Portuguese minimum wage 2024
  location: 'continent'
});

console.log('=== Minimum Wage Worker ===');
console.log(\`Gross Income: €\${minimumWageWorker.grossIncome.toFixed(2)}\`);
console.log(\`Tax: €\${minimumWageWorker.IrsTax.toFixed(2)}\`);
console.log(\`Social Security: €\${minimumWageWorker.socialSecurityTax.toFixed(2)}\`);
console.log(\`Net Salary: €\${minimumWageWorker.net.salary.toFixed(2)}\`);
console.log(\`Yearly Net: €\${minimumWageWorker.yearlyNetSalary.toFixed(2)}\`);`} />

## Family Situations

### Single Parent with 2 Children

<RunCode defaultCode={`const singleParent = simulateDependentWorker({
  income: 1500,
  married: false,
  numberOfDependents: 2,
  location: 'continent'
});

console.log('=== Single Parent (2 children) ===');
console.log(\`Monthly Net: €\${singleParent.net.salary.toFixed(2)}\`);

// Compare with same income but no dependents
const noDependents = simulateDependentWorker({ 
  income: 1500, 
  location: 'continent' 
});

console.log(\`Tax Savings from dependents: €\${(noDependents.IrsTax - singleParent.IrsTax).toFixed(2)}\`);`} />

### Married Dual Income (2 Holders)

<RunCode defaultCode={`// Spouse 1
const spouse1 = simulateDependentWorker({
  income: 1400,
  married: true,
  numberOfHolders: 2,
  numberOfDependents: 1, // One child shared
  location: 'continent'
});

// Spouse 2
const spouse2 = simulateDependentWorker({
  income: 1100,
  married: true,
  numberOfHolders: 2,
  numberOfDependents: 0, // Child already counted in spouse1
  location: 'continent'
});

const totalHouseholdNet = spouse1.net.salary + spouse2.net.salary;
console.log('=== Married Dual Income ===');
console.log(\`Spouse 1 Net: €\${spouse1.net.salary.toFixed(2)}\`);
console.log(\`Spouse 2 Net: €\${spouse2.net.salary.toFixed(2)}\`);
console.log(\`Total Household Net: €\${totalHouseholdNet.toFixed(2)}\`);`} />

## Period-based Examples

### Tax Period Transitions

<RunCode defaultCode={`// First period of 2025
const firstPeriod = simulateDependentWorker({
  income: 1800,
  period: "2025-01-01_2025-07-31", // January 1 - July 31
  location: 'continent'
});

// Second period of 2025
const secondPeriod = simulateDependentWorker({
  income: 1800,
  period: "2025-08-01_2025-09-30", // August 1 - September 30
  location: 'continent'
});

// Third period of 2025
const thirdPeriod = simulateDependentWorker({
  income: 1800,
  period: "2025-10-01_2025-12-31", // October 1 - December 31
  location: 'continent'
});

console.log('=== Tax Period Comparison ===');
console.log(\`First Period Net: €\${firstPeriod.net.salary.toFixed(2)}\`);
console.log(\`Second Period Net: €\${secondPeriod.net.salary.toFixed(2)}\`);
console.log(\`Third Period Net: €\${thirdPeriod.net.salary.toFixed(2)}\`);`} />

These examples demonstrate the flexibility and comprehensive nature of Saldo for various Portuguese tax scenarios.
