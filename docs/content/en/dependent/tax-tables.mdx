# Tax Tables Explorer

This interactive page lets you explore every Portuguese withholding table used by Saldo. You can filter and search tables to understand tax brackets and rates across regions and periods.

> These are the retention tables for the **dependent worker** simulator. Independent worker calculations use separate progressive tax ranks (2023-2025) and are not shown here.

## Features

- **Real bracket data**: official income ranges, rates, and deductions
- **Multi-filter support**: region, period, marital status, dependents, disability, and holders
- **Fast search**: by table name, description, and situation
- **Expandable details**: click a row to inspect bracket-level values
- **Raw data access**: copy JSON or open the source file on GitHub

## Available Data

The tax tables include data for:
- **Tax years**: all years bundled with the installed package
- **Regions**: continent, Azores, and Madeira
- **Date ranges**: one or more validity intervals per year
- **Table types**: TABLE1 through TABLE7 for different family situations

## Tax Bracket Structure

Each tax bracket contains:
- **Income Range**: Minimum or maximum income threshold for the bracket
- **Marginal Rate**: The tax rate applied to income in this bracket
- **Deduction**: Base deduction amount for this bracket
- **Var1 Deduction**: Variable deduction component 1
- **Var2 Deduction**: Variable deduction component 2
- **Dependent Additional**: Additional deduction per dependent
- **Effective Rate**: The effective monthly tax rate for this bracket

## Usage

Use the filters above the table to narrow results by region, period, and fiscal situation. You can also search by description. Click any row to expand and inspect bracket details.

## Data Source

The data comes from official Portuguese withholding tables and is loaded from the `taxTablesData` manifest. This keeps the page aligned with the exact tables used by the calculation engine.
