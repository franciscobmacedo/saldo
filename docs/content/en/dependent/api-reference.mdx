# API Reference

Dependent worker APIs for Portuguese payroll. For self-employed calculations, see the [independent worker API](/independent/api-reference).

## Main Function

### `simulateDependentWorker(options)`

Calculates Portuguese dependent worker taxes and net salary using the official retention tables.

#### Parameters

```typescript
interface SimulateDependentWorkerOptions {
  income: number;
  married?: boolean;
  disabled?: boolean;
  partnerDisabled?: boolean;
  location?: LocationT;
  numberOfHolders?: number | null;
  numberOfDependents?: number | null;
  numberOfDependentsDisabled?: number | null;
  period?: PeriodT;
  socialSecurityTaxRate?: number;
  twelfths?: Twelfths;
  lunchAllowanceDailyValue?: number;
  lunchAllowanceMode?: "cupon" | "salary";
  lunchAllowanceDaysCount?: number;
}
```

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `income` | `number` | **Required** | Monthly gross income in euros |
| `married` | `boolean` | `false` | Whether the person is married |
| `disabled` | `boolean` | `false` | Whether the person has a disability |
| `partnerDisabled` | `boolean` | `false` | Whether the spouse has a disability |
| `location` | `LocationT` | `'continent'` | Tax region: `'continent'`, `'azores'`, or `'madeira'` |
| `numberOfHolders` | `number \| null` | `null` | Number of income holders in marriage (1 or 2) |
| `numberOfDependents` | `number \| null` | `null` | Number of dependents |
| `numberOfDependentsDisabled` | `number \| null` | `null` | Number of dependents with disabilities |
| `period` | `PeriodT` | `"2025-01-01_2025-07-31"` | Tax calculation period in format "YYYY-MM-DD_YYYY-MM-DD" |
| `socialSecurityTaxRate` | `number` | `0.11` | Social security tax rate (11%) |
| `twelfths` | `Twelfths` | `Twelfths.TWO_MONTHS` | Holiday allowance distribution |
| `lunchAllowanceDailyValue` | `number` | `10.2` | Daily lunch allowance amount in euros |
| `lunchAllowanceMode` | `"cupon" \| "salary"` | `"cupon"` | Type of lunch allowance |
| `lunchAllowanceDaysCount` | `number` | `22` | Working days per month for lunch allowance |

#### Return Value

```typescript
interface DependentWorkerResult {
  taxableIncome: number;
  tax: number;
  socialSecurity: number;
  socialSecurityTax: number;
  gross: {
    monthly: number;  // Total gross salary per month
    yearly: number;   // Gross salary for the year (14 months)
  };
  net: {
    base: number;    // Net income from base + twelfths (excludes lunch allowance)
    salary: number;  // Total net salary: base + lunch allowance
    yearly: number;  // Net salary for the year
  };
  lunchAllowance: LunchAllowanceResult;  // Calculated values only
  bracket: { ... };
  taxRetentionTable: { ... };
}
```

| Property | Type | Description |
|----------|------|-------------|
| `taxableIncome` | `number` | Monthly income subject to tax |
| `tax` | `number` | Monthly tax amount |
| `socialSecurity` | `number` | Monthly social security contribution |
| `socialSecurityTax` | `number` | Social security tax rate applied |
| `gross.monthly` | `number` | Total monthly gross income including allowances |
| `gross.yearly` | `number` | Annual gross salary (14 months) |
| `net.base` | `number` | Net income from base salary + twelfths (Rendimento líquido). Excludes lunch allowance |
| `net.salary` | `number` | Total monthly net salary after deductions (Salário líquido = net.base + lunch allowance) |
| `net.yearly` | `number` | Annual net salary (14 months) |
| `lunchAllowance` | `LunchAllowanceResult` | Calculated lunch allowance: gross, net, taxable, taxFree |
| `bracket` | `object` | Tax bracket information with rates and deductions |
| `taxRetentionTable` | `object` | Complete tax retention table data |

#### Example

<RunCode defaultCode={`
const result = simulateDependentWorker({
  income: 1500,
  married: true,
  numberOfHolders: 1,
  numberOfDependents: 2,
  location: 'continent',
  lunchAllowanceDailyValue: 8.5,
  lunchAllowanceMode: 'cupon',
  lunchAllowanceDaysCount: 22,
  twelfths: Twelfths.TWO_MONTHS
});

// Doutor Finanças-style breakdown:
console.log(\`Net Income: €\${result.net.base.toFixed(2)}\`);
console.log(\`Lunch Allowance: €\${result.lunchAllowance.gross.toFixed(2)}\`);
console.log(\`Net Salary: €\${result.net.salary.toFixed(2)}\`);`} />

## Types and Enums

### `LocationT`

```typescript
type LocationT = "continent" | "azores" | "madeira";
```

### `Twelfths`

```typescript
enum Twelfths {
  NONE = 0,           // No holiday allowances
  ONE_HALF_MONTH = 0.5, // 1x50% - One allowance at 50% (half month)
  ONE_MONTH = 1,       // 2x50% - Two allowances at 50% each (one month total)
  TWO_MONTHS = 2       // 2x100% - Two allowances at 100% each (two months total)
}
```

### `LunchAllowanceResult`

The result contains calculated lunch allowance values only (no input data):

```typescript
interface LunchAllowanceResult {
  gross: number;    // Gross monthly amount
  net: number;     // Net amount after tax/SS on taxable portion
  taxable: number; // Portion subject to tax and social security
  taxFree: number; // Portion exempt from tax and social security
}
```

| Property | Type | Description |
|----------|------|-------------|
| `gross` | `number` | Total monthly lunch allowance |
| `net` | `number` | Net amount after tax and social security on taxable portion |
| `taxable` | `number` | Portion subject to tax |
| `taxFree` | `number` | Tax-free portion |

**Tax Rules**:
- **Meal vouchers** (`"cupon"`): Tax-free up to €10.20/day
- **Cash allowance** (`"salary"`): Tax-free up to €6.00/day

### `SituationCodesT`

```typescript
type SituationCodesT =
  | "TABLE1_SINGLE_OR_MARRIED_2_HOLDERS"      // Single or married with 2 holders
  | "TABLE2_SINGLE_ONE_OR_MORE_DEPENDENTS"     // Single with dependents
  | "TABLE3_MARRIED_1_HOLDER"                    // Married, single holder
  | "TABLE4_SINGLE_OR_MARRIED_2_HOLDERS_NO_DEPENDENTS_PERSON_WITH_DISABILITY"  // Single or married 2 holders + disability
  | "TABLE5_SINGLE_ONE_OR_MORE_DEPENDENTS_PERSON_WITH_DISABILITY"     // Single with dependents + disability
  | "TABLE6_MARRIED_2_HOLDERS_ONE_OR_MORE_DEPENDENTS_PERSON_WITH_DISABILITY"    // Married 2 holders with dependents + disability
  | "TABLE7_MARRIED_1_HOLDER_PERSON_WITH_DISABILITY";    // Married single holder + disability
```

### `Situation`

```typescript
interface Situation {
  code: SituationCodesT;
  description: string;
  conditions: Condition[];
}
```

### `Condition`

```typescript
interface Condition {
  married: boolean;
  dependents: boolean;
  disabled: boolean;
  partnerDisabled: boolean;
  description: string;
  numberOfHolders?: number | null;
}
```

### `Bracket`

```typescript
interface Bracket {
  signal: "max" | "min";                    // Whether this is a maximum or minimum bracket
  limit: number;                            // Income limit for this bracket
  max_marginal_rate: number;                // Maximum marginal tax rate
  deduction: number;                        // Standard deduction amount
  var1_deduction: number;                   // Variable deduction 1
  var2_deduction: number;                   // Variable deduction 2
  dependent_aditional_deduction: number;   // Additional deduction per dependent
  effective_mensal_rate: number;            // Effective monthly tax rate
}
```

### `TaxRetentionTable`

```typescript
interface TaxRetentionTable {
  situation: string;                       // Situation code
  description: string;                     // Human-readable description
  brackets: Bracket[];                      // Array of all tax brackets
  dependent_disabled_addition_deduction?: number; // Additional deduction for disabled dependents
}
```

### `RetentionPathsSchema`

```typescript
class RetentionPathsSchema {
  public identifier: string;

  constructor(
    period: PeriodT,
    location: LocationT,
    situationCode: SituationCodesT,
    year: number | string
  );

  get path(): string; // Returns the identifier
}
```

Constructs identifiers in the format `"2025/continent/2025-01-01_2025-07-31/TABLE1_SINGLE_OR_MARRIED_2_HOLDERS"`.

## Utility Functions

### `SituationUtils.getSituation()`

```typescript
SituationUtils.getSituation(
  married: boolean,
  disabled: boolean,
  partnerDisabled: boolean,
  numberOfHolders?: number | null,
  numberOfDependents?: number
): Situation | undefined
```

Determines the appropriate tax situation based on personal circumstances.

### `SituationUtils.getSituationFromCode()`

```typescript
SituationUtils.getSituationFromCode(code: string): Situation | undefined
```

Retrieves a tax situation by its code.

## Validation Functions

The library includes internal validation functions that are automatically called:

- `validateNumberOfHolders(numberOfHolders)`
- `validateMarriedAndNumberOfHolders(married, numberOfHolders)`
- `validateDependents(numberOfDependents, numberOfDependentsDisabled)`

## Error Handling

<RunCode defaultCode={`// Invalid situation
try {
  simulateDependentWorker({
    income: 1500,
    married: true,
    // Missing numberOfHolders - required for married
  });
} catch (error) {
  console.error('Error:', error.message); // "'numberOfHolders' is required for married workers"
}

// Invalid bracket
try {
  simulateDependentWorker({
    income: 999999, // Extremely high income
    location: 'continent'
  });
} catch (error) {
  console.error('Error:', error.message); // "No bracket found for salary..."
}`} />

## Advanced Usage

### Period-based Calculations

<RunCode defaultCode={`const result = simulateDependentWorker({
  income: 1500,
  period: "2025-01-01_2025-07-31",  // First period of 2025
  location: 'continent'
});

console.log('=== Period-based Calculation ===');
console.log(\`Net Salary: €\${result.net.salary.toFixed(2)}\`);
console.log(\`Yearly Net: €\${result.net.yearly.toFixed(2)}\`);`} />

### Custom Social Security Rates

<RunCode defaultCode={`const result = simulateDependentWorker({
  income: 1500,
        socialSecurityTaxRate: 0.11, // 11% (default)
  location: 'continent'
});

console.log('=== Custom Social Security Rate ===');
console.log(\`Social Security: €\${result.socialSecurity.toFixed(2)}\`);
console.log(\`Rate Applied: \${(result.socialSecurityTax * 100).toFixed(1)}%\`);`} />

### Disability Benefits

<RunCode defaultCode={`// Worker with disability
const workerDisability = simulateDependentWorker({
  income: 1500,
  disabled: true,
  location: 'continent'
});

// Partner with disability (extra deduction)
const partnerDisability = simulateDependentWorker({
  income: 1500,
  married: true,
  numberOfHolders: 1,
  partnerDisabled: true, // €135.71 extra deduction
  location: 'continent'
});

// Dependent with disability (extra deduction per dependent)
const dependentDisability = simulateDependentWorker({
  income: 1500,
  numberOfDependents: 2,
  numberOfDependentsDisabled: 1,
  location: 'continent'
});

console.log('=== Disability Benefits Comparison ===');
console.log(\`Worker Disability - Net: €\${workerDisability.net.salary.toFixed(2)}\`);
console.log(\`Partner Disability - Net: €\${partnerDisability.net.salary.toFixed(2)}\`);
console.log(\`Dependent Disability - Net: €\${dependentDisability.net.salary.toFixed(2)}\`);`} />
