# Getting Started

Welcome to Saldo's **dependent worker** guide. If you need the self-employed flow, jump to the [independent worker docs](/independent/get-started).

## Installation

```bash
npm install saldo
# or
pnpm add saldo
```

## Understanding the Result

```typescript
interface DependentWorkerResult {
  taxableIncome: number;
  IrsTax: number;
  socialSecurityTax: number;
  socialSecurityTaxRate: number;
  gross: { monthly: number; yearly: number };
  net: { base: number; salary: number; yearly: number };
  lunchAllowance: LunchAllowanceResult;  // gross, net, taxable, taxFree
  bracket: Bracket;
  taxRetentionTable: TaxRetentionTable;
}
```

## Key Parameters

- **`income`**: Monthly gross income in euros
- **`married`** / **`numberOfHolders`**: Required together for married workers (1 or 2)
- **`disabled`**, **`partnerDisabled`**, **`numberOfDependents`**, **`numberOfDependentsDisabled`**
- **`location`**: `'continent' | 'azores' | 'madeira'` (default `'continent'`)
- **`period`**: `"2025-01-01_2025-07-31"`, `"2025-08-01_2025-09-30"`, `"2025-10-01_2025-12-31"`
- **`socialSecurityTaxRate`**: Default `0.11`
- **`twelfths`**: `Twelfths.NONE | ONE_HALF_MONTH | ONE_MONTH | TWO_MONTHS` (default `TWO_MONTHS`)
- **`lunchAllowanceDailyValue`**, **`lunchAllowanceMode`** (`"cupon"` or `"salary"`), **`lunchAllowanceDaysCount`**

## Working with Types

```typescript
import { 
  simulateDependentWorker, 
  LocationT, 
  Twelfths, 
  DependentWorkerResult,
  SimulateDependentWorkerOptions,
  LunchAllowanceResult
} from 'saldo';

// Type-safe dependent worker calculation
const dependentOpts: SimulateDependentWorkerOptions = {
  income: 1500,
  location: 'continent',
};
const dependentResult: DependentWorkerResult = simulateDependentWorker(dependentOpts);
```
