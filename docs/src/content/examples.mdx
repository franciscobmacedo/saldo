# Examples

Real-world examples showing how to use Saldo TS for different scenarios.

## Basic Calculations

### Minimum Wage Worker

```typescript
import { simulateDependentWorker } from 'saldo-ts';

const minimumWageWorker = simulateDependentWorker({
  income: 760, // Portuguese minimum wage 2024
  location: 'continente'
});

console.log('=== Minimum Wage Worker ===');
console.log(`Gross Income: €${minimumWageWorker.grossIncome.toFixed(2)}`);
console.log(`Tax: €${minimumWageWorker.tax.toFixed(2)}`);
console.log(`Social Security: €${minimumWageWorker.socialSecurity.toFixed(2)}`);
console.log(`Net Salary: €${minimumWageWorker.netSalary.toFixed(2)}`);
console.log(`Yearly Net: €${minimumWageWorker.yearlyNetSalary.toFixed(2)}`);
```

### Average Worker

```typescript
const averageWorker = simulateDependentWorker({
  income: 1200,
  location: 'continente'
});

console.log('=== Average Worker ===');
console.log(`Monthly Net: €${averageWorker.netSalary.toFixed(2)}`);
console.log(`Yearly Net: €${averageWorker.yearlyNetSalary.toFixed(2)}`);
console.log(`Effective Tax Rate: ${(averageWorker.tax / averageWorker.taxableIncome * 100).toFixed(1)}%`);
```

### Married Dual Income (2 Holders)

```typescript
// Spouse 1
const spouse1 = simulateDependentWorker({
  income: 1400,
  married: true,
  numberOfHolders: 2,
  numberOfDependents: 1, // One child shared
  location: 'continente'
});

// Spouse 2
const spouse2 = simulateDependentWorker({
  income: 1100,
  married: true,
  numberOfHolders: 2,
  numberOfDependents: 0, // Child already counted in spouse1
  location: 'continente'
});

const totalHouseholdNet = spouse1.netSalary + spouse2.netSalary;
console.log('=== Married Dual Income ===');
console.log(`Spouse 1 Net: €${spouse1.netSalary.toFixed(2)}`);
console.log(`Spouse 2 Net: €${spouse2.netSalary.toFixed(2)}`);
console.log(`Total Household Net: €${totalHouseholdNet.toFixed(2)}`);
```

## Date Range Examples

### Tax Year Transitions

```typescript
// Early year rates
const earlyYear = simulateDependentWorker({
  income: 1800,
  dateStart: new Date(2024, 0, 1), // January 1
  dateEnd: new Date(2024, 5, 30),  // June 30
  location: 'continente'
});

// Late year rates  
const lateYear = simulateDependentWorker({
  income: 1800,
  dateStart: new Date(2024, 6, 1),  // July 1
  dateEnd: new Date(2024, 11, 31), // December 31
  location: 'continente'
});

console.log('=== Tax Year Comparison ===');
console.log(`H1 Net: €${earlyYear.netSalary.toFixed(2)}`);
console.log(`H2 Net: €${lateYear.netSalary.toFixed(2)}`);
```

These examples demonstrate the flexibility and comprehensive nature of Saldo TS for various Portuguese tax scenarios.
