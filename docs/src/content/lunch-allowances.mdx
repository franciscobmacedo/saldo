# Lunch Allowances

Lunch allowances (subsídio de alimentação) are a common benefit in Portuguese employment. Understanding how they're taxed is crucial for accurate salary calculations.

## Overview

Lunch allowances in Portugal have specific tax treatment:
- **Tax-free portion**: Up to certain daily limits
- **Taxable portion**: Amount exceeding the limits
- **Social security**: May be exempt depending on the type

## Types of Lunch Allowances

### Meal Vouchers (Cupões de Refeição)

Daily tax-free limit: **€10.20**

```typescript
const result = simulateDependentWorker({
  income: 1500,
  lunchAllowanceDailyValue: 8.50,
  lunchAllowanceMode: "cupon",
  lunchAllowanceDaysCount: 22
});

console.log(`Monthly lunch allowance: €${result.lunchAllowance.monthlyValue}`);
console.log(`Tax-free portion: €${result.lunchAllowance.taxFreeMonthlyValue}`);
console.log(`Taxable portion: €${result.lunchAllowance.taxableMonthlyValue}`);
```

### Cash Allowance (Em Dinheiro)

Daily tax-free limit: **€6.00**

```typescript
const result = simulateDependentWorker({
  income: 1500,
  lunchAllowanceDailyValue: 7.50,
  lunchAllowanceMode: "salary",
  lunchAllowanceDaysCount: 21
});

console.log(`Monthly lunch allowance: €${result.lunchAllowance.monthlyValue}`);
console.log(`Tax-free portion: €${result.lunchAllowance.taxFreeMonthlyValue}`);
console.log(`Taxable portion: €${result.lunchAllowance.taxableMonthlyValue}`);
```

## Parameters

When using lunch allowances, you can specify these parameters:

- **`lunchAllowanceDailyValue`**: Daily allowance amount in euros
- **`lunchAllowanceMode`**: Type of allowance (`"cupon"` or `"salary"`)
- **`lunchAllowanceDaysCount`**: Number of working days per month

## Common Scenarios

### Standard Meal Vouchers

Most common scenario with meal vouchers below the tax-free limit:

```typescript
const result = simulateDependentWorker({
  income: 1200,
  lunchAllowanceDailyValue: 7.50,
  lunchAllowanceMode: "cupon",
  lunchAllowanceDaysCount: 22
});

// Monthly allowance: €165 (7.50 * 22)
// Tax-free: €165 (fully tax-free as it's below €10.20/day)
// Taxable: €0
console.log(`Net benefit: €${result.lunchAllowance.taxFreeMonthlyValue}`);
```

### High-Value Meal Vouchers

Meal vouchers exceeding the tax-free limit:

```typescript
const result = simulateDependentWorker({
  income: 1200,
  lunchAllowanceDailyValue: 12.00,
  lunchAllowanceMode: "cupon",
  lunchAllowanceDaysCount: 22
});

// Monthly allowance: €264 (12.00 * 22)
// Tax-free: €224.40 (10.20 * 22)
// Taxable: €39.60 ((12.00 - 10.20) * 22)
console.log(`Total allowance: €${result.lunchAllowance.monthlyValue}`);
console.log(`Taxable portion: €${result.lunchAllowance.taxableMonthlyValue}`);
```

### Cash Lunch Allowance

Cash allowance with lower tax-free limit:

```typescript
const result = simulateDependentWorker({
  income: 1500,
  lunchAllowanceDailyValue: 8.00,
  lunchAllowanceMode: "salary",
  lunchAllowanceDaysCount: 21
});

// Monthly allowance: €168 (8.00 * 21)
// Tax-free: €126 (6.00 * 21)
// Taxable: €42 ((8.00 - 6.00) * 21)
console.log(`Tax-free portion: €${result.lunchAllowance.taxFreeMonthlyValue}`);
console.log(`Taxable portion: €${result.lunchAllowance.taxableMonthlyValue}`);
```

### No Lunch Allowance

Using the default (no allowance):

```typescript
const result = simulateDependentWorker({
  income: 1500
  // No lunch allowance parameters specified
});

console.log(`Lunch allowance: €${result.lunchAllowance.monthlyValue}`); // €0
```

## Tax Implications

### Income Tax

- **Tax-free portion**: Not included in taxable income
- **Taxable portion**: Added to monthly taxable income
- **Impact**: Higher total allowances may push you into higher tax brackets

### Social Security

- **Meal vouchers**: Generally exempt from social security contributions
- **Cash allowances**: May be subject to social security (implementation dependent)

## Regional Considerations

Lunch allowance tax treatment is generally consistent across:
- Mainland Portugal (continente)
- Azores (acores)  
- Madeira (madeira)

However, always verify current regulations as they may change.

## Best Practices

### For Employees

1. **Understand your allowance type**: Vouchers vs. cash have different limits
2. **Track daily amounts**: Ensure you're within tax-free limits
3. **Consider total compensation**: Balance base salary vs. allowances

### For Payroll Systems

1. **Validate daily limits**: Ensure calculations use correct tax-free amounts
2. **Handle partial months**: Adjust days count for holidays/absences
3. **Stay updated**: Tax-free limits may change annually
