# Saldo TS

**Portuguese Tax Calculation Library for TypeScript**

Saldo TS is a comprehensive TypeScript library for calculating Portuguese dependent worker taxes, social security contributions, and net salary calculations. It handles complex tax scenarios including different marital statuses, dependents, disabilities, and regional variations across mainland Portugal, Azores, and Madeira.

## Features

- **Accurate Tax Calculations**: Based on official Portuguese tax retention tables
- **Regional Support**: Handles tax differences between mainland Portugal, Azores, and Madeira
- **Complex Scenarios**: Supports married/single status, dependents, disability benefits
- **Lunch Allowances**: Calculates taxable and tax-free portions of meal allowances
- **Holiday Allowances**: Handles 13th and 14th month salary distributions
- **Type Safety**: Full TypeScript support with comprehensive type definitions
- **Flexible**: Configurable for different calculation periods and parameters

## Quick Start

```typescript
import { simulateDependentWorker } from 'saldo-ts';

// Basic calculation for single person
const result = simulateDependentWorker({
  income: 1500, // Monthly gross income
  location: 'continente',
  dateStart: new Date(2024, 0, 1),
  dateEnd: new Date(2024, 11, 31)
});

console.log(`Net Salary: €${result.net_salary.toFixed(2)}`);
console.log(`Tax: €${result.tax.toFixed(2)}`);
console.log(`Social Security: €${result.social_security.toFixed(2)}`);
```

## Core Concepts

### Tax Situations

The library automatically determines the appropriate tax table based on:
- **Marital Status**: Single or married
- **Number of Income Holders**: 1 or 2 (for married couples)
- **Dependents**: Number of dependent children/relatives
- **Disability Status**: Worker or partner disability benefits

### Supported Locations

- **`continente`**: Mainland Portugal
- **`acores`**: Azores region
- **`madeira`**: Madeira region

Each location has different tax rates and may have specific calculation rules.

### Time Periods

Tax calculations are based on specific date ranges as tax tables change over time. The library includes historical data for accurate calculations across different periods.
